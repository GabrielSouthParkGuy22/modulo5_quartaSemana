1.O que são objetos em JavaScript?
R: objetos são um tipo de dados estruturado em JS, podendo conter vários diferentes tipos de dados armazenados, por exemplo:

const myObj = {
  name: "Gabriel",
  middleName: "Oliveira"
  age: 19,
  height: 1.75,
}
